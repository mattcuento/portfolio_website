DRC Tram Web Application

+

HTML | CSS | Javascript | Firebase | Mapbox | Angular

As a member of Quality of Life+, I developed a web application to revamp Cal Poly's Disability Resource Center transportation system along with a team of programmers and industrial engineers. The web application contained views for students, administration, and drivers. The application provided students a way to track their rides and schedule new ones on the go. In addition, drivers could track their next pickups and administrators could oversee all operations from a higher access level. The application featured:

  • Live Location Tracking
  • Routing Algorithm
  • Differentiatied Permissions
  • Styled Map of Stops
  • User Registration/Eligibility Dates
